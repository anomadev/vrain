### Definición
PHP es un lenguaje de programación de propósito general, ampliamente usando especialmente para el desarrollo web.
El nombre original era Personal Home Page, pero hoy se entiende como "PHP: Hypertext Preprocessor". PHP se ejecuta en el servidor (server-side) y genera contenido dinámico que se envía al navegador. 
Es gratuito, de código abierto, y compatible con la mayoría de los sistemas operativos y servidores web. Fue diseñado para que se fácil de aprender para principiantes, pero también ofrece características avanzadas para expertos.
### Conceptos Clave
- **Lenguaje del lado del servidor:** PHP corre en el servidor, no en el navegador.
- **Código embebido en HTML:** Puedes mezclar PHP con HTML en el mismo archivo.
- **Gratuito y de código abierto:** Cualquier persona puede usar y modificar PHP libremente.
- **Multiparadigma:** Soporta programación estructurada, orientada a objetos y funcional.
- **Extensa comunidad:** Hay toneladas de documentación, frameworks y recursos disponibles.
- **Amplio soporte:** PHP puede interactuar con casi cualquier base de datos y servidor.
### Ejemplos prácticos

```php
<!DOCTYPE html>
<html>
<body>

<h1>Mi primer script PHP</h1>

<?php
echo "¡Hola mundo desde PHP!";
?>

</body>
</html>
```

## Evolución e Historia de PHP
- **1994**: Rasmus Lerdorf crea PHP como un conjunto de scripts CGI para rastrear visitantes en su currículum en línea.
- **1995**: Se publica PHP/FI (Personal Home Page/ Forms Interpreter).
- **1997-1998**: PHP 3.0 es rediseñado y extendido por Andi Gutmans y Zeev Suraski
- **1999**: Nace PHP 4.0 utilizando el Zend Engine para mejorar el rendimiento. 
- **2004**: Sale PHP 5, con soporte fuerte para Programación Orientada a Objetos.
- **2015**: Llega PHP 7, omitiendo PHP 6, con mejora masivas en rendimiento (doble de rápido).
- **2020**: Aparece PHP 8, que introduce tipado fuerte, atributos, expresiones match y el compilador JIT
### Conceptos Clave
- **Zend Engine**: Motor principal que ejecuta PHP
- **Orientación a objetos**: PHP 5 introduce clases, herencia, interfaces, etc.
- **Gran salto de rendimiento**: PHP 7 optimiza severamente la velocidad y uso de memoria.
- **PHP 8**: Moderniza PHP con tipado fuerte y nuevas construcciones de lenguaje.
## Versiones de PHP y sus características
#### PHP 3 (1998)
- Primera versión popular
- Soporte de extensiones externas
#### PHP 4 (2000)
- Introduce Zend Engine 1.0
- Mejor manejo de sesiones
- Aún limitado en Programación Orientada a Objetos (POO)
#### PHP 5 (2004)
- Zend Engine 2.0
- Soporte robusto de POO
- Manejo de excepciones
- Introducción de PDO (PHP Data Objects) para acceso a bases de datos
#### PHP 7 (2015)
- Zend Engine 3.0
- Hasta 2x más rápido que PHP 5
- Tipado escalar y retorno de tipos
- Operador de fusión de null (`??`)
#### PHP 8 (2020)
- Compilador JIT (Just In Time)
- Tipos de unión (Union Types)
- Expresiones `match` (similar a `switch` ([[4.3 switch]]), pero más potente)
- Atributos (para reemplazar anotaciones PHPDoc)
- Nuevas funciones como `str_contains`, `str_starts_with`, etc.
