En la sintaxis básica de PHP, el código se delimita mediante etiquetas de apertura y cierre específicas como `<?php ... ?>` (y su forma corta `<?= ... ?>)` mientras que todo lo que esté fuera de ella se trata como HTML estático y es ignorado por el intérprete de PHP. Cada instrucción debe terminar en punto y coma `;`, aunque el cierre de un bloque `?>` lo incluye implícitamente. Para documentar el código se pueden usar comentarios de línea con `//` o `#`, así como comentarios de bloque con `/* ... */`.
### Definición
La sintaxis de PHP determina cómo escribir código PHP embebido en documentos (normalmente HTML), especificando:
- Las etiquetas que delimitan el inicio y fin de ejecución de código (`<?php ?>` y `<?= ?>`).
- La forma de escapar de HTML para mezclar contenido dinámico y estático.
- La separación de instrucciones mediante punto y coma `;` 
- Los estilos de comentario soportados (`//`, `#`, `/* ... */`).
### Conceptos clave
- Etiquetas PHP:
	- Apertura: `<?php` (requiere un espacio/tab/salto de línea después).
	- Cierre: `?>`.
	- `echo` corto: `<?=` es equivalente a `<?php echo`.
- Escapando HTML:
	- Todo lo fuera de `<?php ... ?>` se imprime tal cual en el navegador.
- Separación de instrucciones:
	- Cada sentencia termina en `;`.
	- Al cerrar un bloque con `?>`, no necesitas `;` tras la última línea.
- Comentarios:
	- Línea: `// comentario` o `# comentario`.
	- Bloque: `/* comentario multilinea */`.
### Ejemplos prácticos
##### Etiquetas y echo corto:
```php
<!DOCTYPE html>
<html>
<body>

  <?php echo "Hola desde PHP"; ?>  <!-- Etiqueta normal -->
  <?= "Hola con echo corto"; ?>     <!-- Equivale a <?php echo … ?> -->
  
</body>
</html>
```
##### Escapando HTML:
```php
<p>Esto se muestra siempre</p>

<?php if ($mostrar): ?>
  <p>Contenido dinámico</p>
<?php endif; ?>

<p>También siempre se muestra</p>
```
##### Separación de instrucciones:
```php
<?php

echo "Línea 1";    // Punto y coma requerido
echo "Línea 2"     // Última línea antes de cerrar, no hace falta ;

?>
```
##### Comentarios:
```php
<?php
// Comentario de línea
# Otro comentario de línea

/*
	Comentario de bloque
*/

echo "Código activo";
?>
```
