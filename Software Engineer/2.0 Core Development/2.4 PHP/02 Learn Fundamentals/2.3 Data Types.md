PHP es un lenguaje de tipado dinámico y débil que asigna automáticamente el tipo de dato de una variable en tiempo de ejecución, sin requerir declaraciones explícitas de tipo. Su sistema de tipos incluye **tipos escalares** (boolean, entero, flotante, cadena), **tipos compuestos** (array, object), **tipos especiales** (null, resource, callable), y mecanismos avanzados como **uniones**, **intersecciones**, **aliases** (mixed, iterable), y **enumeraciones**. Esta flexibilidad facilita la programación rápida y el manejo de datos heterogéneos sin tener que definir manualmente cada estructura.
### Definición
Los tipos de datos en PHP son las categorías que definen el conjunto de valores que una variable puede contener y las operaciones válidas sobre ellos. PHP distingue principalmente:
- Tipos escalares: `bool`, `int`, `float`, `string`.
- Tipos compuestos: `array`, `object`.
- Tipos especiales: `NULL`, `resource`, `callable`, `iterable`.
- Tipos especiales de control: `void`, `never`, `self`, `parent`, `static` y los **singleton types** `true/false`.
### Conceptos clave
- **Tipado dinámico y débil**: PHP infiere el tipo en tiempo de ejecución y convierte valores automáticamente cuando es necesario.
- **Tipos escalares**:
	- `Boolean`: true o false
	- `Integer`: números enteros en decimal, hexadecimal, octal, o binario
	- `Float`: números con parte fraccionaria o notación científica
	- `String`: secuencias de caracteres, admiten comillas simples, dobles, `heredoc` y `nowdoc`
- **Tipos compuestos**:
	- `Array`: colección indexada u asociativa, elementos de cualquier tipo ([[3.1 Arrays]])
	- `Object`: instancias de clases definidas por el usuario ([[11.1 Classes and Objects]])
- **Tipos especiales**:
	- `NULL`: valor único que denota "*sin valor*"
	- `Resource`: referencias a recursos externos (archivos, conexiones, etc.)
	- `Callable`: referencias a funciones o métodos invocables
	- `Iterable`: alias de `Traversable|array`, válido en bucles `foreach` ([[5.0 Loops]])
- **Uniones e intersecciones**:
	- `Union types` (`T|U`): acepta valores de cualquiera de los tipos listados
	- `Intersection types` (`T&U`): acepta solo valores que cumplan todos los tipos simultáneamente.
- **Aliases de tipo**:
	- `mixed`: equivalente a `object|resource|array|string|float|int|bool|null`
	- `iterable`: alias de `Traversable|array`
### Ejemplos prácticos
##### `Integer` en diferentes bases
```php
<?php

$a = 12345; // decimal
$b = 0x1B;  // hexadecimal (27 decimal)
$c = 0321;  // octal (209 decimal)
$d = 0b11;  // binario (3 decimal)

?>
```
##### `Float` y notación científica
```php 
<?php

$x = 1.234; // flotante
$y = 1.2e3; // 1200
$z = 7E-5;  // 0.00007

?>
```
##### `Array` asociativo y multidimensional
```php
<?php

$salarios = ["Ana" => 3400, "Luis" => 4300];
echo $salarios["Luis"]; // 43000

$notas = [
	"Carlos" => ["matematicas" => 75, "historia" => 85],
	"Marta" => ["matematicas" => 88, "historia" => 92]
];
echo $notas["Marta"]["historia"]; // 92

?>
```
##### `Nullable` y unión de tipos (PHP 8)
```php
<?php

function buscar(int|string $id): ?array {
	// ... devuelve array o null
}

?>
```
##### `Iterable` en un `foreach`
```php
<?php

function procesar(iterable $datos) {
	foreach ($datos as $valor)
		echo $valor;
	}
}

procesar([1,2,3]); // con array
procesar(new ArrayIterator([4,5,6])); // con objeto Traversable

?>
```
