En PHP, un array es una estructura de datos que actúa como un mapa ordenado, asociando claves y valores. Esta versatilidad permite que los `arrays` funcionen como listas, pilas, colas, diccionarios, entre otros. Las claves pueden ser enteros o cadenas, y los valores pueden ser de cualquier tipo, incluyendo otros `arrays`, lo que permite la creación de estructuras multidimensionales.
### Definición 
Un `array` en PHP es una colección ordenada de pares clave-valor. Las claves pueden ser enteros o cadenas, y los valores pueden ser de cualquier tipo. PHP permite la creación de `arrays` de diversas formas:
- Array indexado ([[3.2 Indexed Arrays]]): las claves son números enteros que PHP asigna automáticamente
```php
<?php
$frutas = array("manzana", "banana", "cereza");
?>
```
- Array asociativo ([[3.3 Associative Arrays]]): Las claves son cadenas definidas por el usuario
```php
<?php
$edades = array("Juan" => 25, "María" => 30, "Pedro" => 28);
?>
```
- Array multidimensional ([[3.4 Multi-dimensional Arrays]]): Un array que contiene uno o más `arrays` como elementos
```php
<?php
$contactos = array(
	"Juan" => array("tel" => "123456789", "email" => "juan@example.com"),
    "María" => array("tel" => "987654321", "email" => "maria@example.com")
);
?>
```
### Conceptos clave
- **Creación de arrays**: Se pueden crear utilizando la función `array()` o la sintaxis corta `[]` (disponible desde PHP 5.4)
```php
<?php
$numeros = array(1, 2, 3);
$letras = ["a", "b", "c"];
?>
```
- **Acceso a elementos**: Se accede a los elementos del array mediante sus claves.
```php
<?php
echo $frutas[0]; // manzana
echo $edades["María"]; // 30
?>
```
- **Modificación y adición de elementos**: Se pueden modificar los valores existentes o añadir nuevos elementos al array.
```php
<?php
$frutas[1] = "pera"; // Modifica "banana" por "pera"
$frutas[] = "naranja"; // Añade "naranja" al final del array
?>
```
- **Eliminación de elementos**: Se puede eliminar un elemento del array utilizando `unset()`.
```php
<?php
unset($frutas[2]); // Elimina "cereza"
?>
```
- **Recorrido de arrays**: Se puede recorrer utilizando estructuras de control como `foreach`.
```php
<?php
foreach ($edades as $nombre => $edad) {
    echo "$nombre tiene $edad años\n";
}
?>
```
### Ejemplos prácticos
##### Array indexado:
```php
<?php
$colores = ["rojo", "verde", "azul"];
echo $colores[1]; // verde
?>
```
##### Array asociativo
```php
<?php
$precios = ["manzana" => 1.2, "banana" => 0.8];
echo $precios["banana"]; // 0.8
?>
```
##### Array multidimensional
```php
<?php
$productos = [
    ["nombre" => "Laptop", "precio" => 1000],
    ["nombre" => "Tablet", "precio" => 500]
];
echo $productos[1]["nombre"]; // Tablet
?>
```
##### Añadir elementos
```php
<?php
$nombres = ["Ana", "Luis"];
$nombres[] = "Carlos";
print_r($nombres);
?>
```
##### Eliminar elementos
```php
<?php
$numeros = [1, 2, 3, 4];
unset($numeros[2]); // Elimina el número 3
print_r($numeros);
?>
```
## Notas
---
- Los `arrays` en PHP son extremadamente flexibles, permitiendo almacenar diferentes tipos de datos y estructuras complejas
- Es importante comprender cómo funcionan las claves de los `arrays`, especialmente al combinar claves numéricas y cadenas.
- El uso adecuado de `arrays` puede simplificar significativamente la gestión de datos en aplicaciones PHP.
