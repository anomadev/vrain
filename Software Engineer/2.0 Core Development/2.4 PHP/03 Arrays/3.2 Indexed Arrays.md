En PHP, un `Indexed Array` es aquel que consta únicamente de valores que PHP indexa automáticamente con claves numéricas secuenciales, comenzando en 0. Estos `arrays` son ideales cuando el orden de los elementos importa y se crean ya sea con `array()` o la sintaxis `[]`. Se accede a sus elementos mediante `$array[indice]`, y nuevos elementos añadidos sin clave explícita reciben el siguiente índice disponible. PHP trata por igual `arrays` indexados y asociativos ([[3.3 Associative Arrays]]) en su implementación interna, pero conceptualmente los indexados son solo una lista lineal de valores. Entre las operaciones comunes están recorrerlos con `foreach` ([[5.0 Loops]]), modificar valores por índice o usar funciones integradas como `count()`, `array_push` y `array_pop` para gestionarlos dinámicamente.
### Definición
Un `Indexed Array` en PHP es un array cuyos elementos solo contienen valores, y PHP les asigna claves numéricas automáticamente a partir de 0, incrementándose en 1 para cada elemento posterior.
- Se crea con el constructo `array(val1, val2)` o con la sintaxis corta `[val1, val2]` (desde PHP 5.4)
- La primera clave siempre es `0`, la siguiente `1`, y así sucesivamente, sin importar el tipo de dato de los valores.
- Internamente, PHP no distingue entre `arrays` indexados y asociativos; ambos se implementan como "`ordered maps`".
### Conceptos clave
- Creación:
	- `array("a", "b", "c")`
	- `["a", "b", "c"]`
- Claves automáticas
	- Sin clave explícita, PHP usa `0`, `1`, `2`, ... basándose en el máximo índice anterior `+1`
- Acceso: Mediante `array(indice)`, p. ej. `$frutas[2]`
- Modificación y adición:
	- Reasignar: `$array[1] = "nuevo"`
	- Añadir: `$array[] = "valor"`
- Recorrido
	- `foreach ($array as $valor) { ... }`
	- `for ($i=0; $i < count($array); $i++) { ... }`
- Funciones útiles: `count()`, `array_push`, `array_pop`, `array_shift()`, `array_unshift()`
- Propiedades:
	- Dinámicos: pueden crear o reducirse en tiempo de ejecución.
	- Pueden mezclarse con claves asociativas, pero siguen siendo indexados en sus valores puros.
### Ejemplos prácticos
##### Creación y visualización
```php
<?php
$colores = ["rojo", "verde", "azul"];
  var_dump($colores);
?>
```
##### Acceso y modificación
```php
<?php
echo $colores[1];   // verde
  $colores[1] = "amarillo";
  echo $colores[1];   // amarillo
?>
```
##### Añadir elementos
```php
<?php
$nombres = ["Ana", "Luis"];
  $nombres[] = "Carlos";
  print_r($nombres);
?>
```
##### Recorrido con `foreach`
```php
<?php
foreach ($nombres as $indice => $valor) {
    echo "nombres[$indice] = $valor\n";
  }
?>
```
##### Recorrido con `for` y `count`
```php
<?php
for ($i = 0; $i < count($colores); $i++) {
	echo "colores[$i] = {$colores[$i]}\n";
}
?>
```

## Notas
---
- Los `Indexed Arrays` son la base de cualquier estructura de datos más compleja en PHP.
- Evitar sobreescribir índices manualmente, a menos que se necesite un control específico.
- Combinar `$array[]` con funciones como `array_map` o `array_filter` facilita el procesamiento de lista.
