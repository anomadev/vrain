Los arrays asociativos en PHP son estructuras de datos que almacenas pares clave-valor, donde las claves son cadenas o enteros definidos por el usuario en lugar de índices numéricos secuenciales. Esta flexibilidad permite acceder a los valores mediante identificadores significativos, facilitando la gestión de datos como directorios telefónicos, configuraciones y colecciones de objetos HTTP. PHP implementa internamente todos los arrays como mapas ordenados, por lo que no diferencia en el motor entre arrays asociativos e indexados ([[3.2 Indexed Arrays]]). Además de la creación y acceso, PHP ofrece funciones integradas para recorrer, buscar, ordenar y transformar estos arrays como: `foreach`, `array_keys`, `array_values`, `array_search`, `asort`, `arsort` y muchas más.
### Definición
Un array asociativo en PHP es un `array` cuyas claves son cadenas (o también enteros definidos manualmente), y los valores pueden ser de cualquier tipo (incluyendo otros arrays u objetos). Se declara usando la sintaxis `array("clave" => valor,...)` o la forma corta `["clave" => valor, ...]` (desde PHP 5.4).
### Conceptos clave
- Mapas ordenados: PHP almacena todos los arrays (indexados o asociativos) internamente como `ordered maps` que retienen el orden de la inserción.
- Declaración:
```php
<?php
$edades = array("Juan" => 25, "María" => 30); 
$precios = ["manzana" => 1.2, "pera" => 0.8];
?>
```
- Acceso a elementos:
```php
<?php
echo $edades["María"]; // 30
?>
```
- Modificación y adición:
```php
<?php
$edades["Pedro"] = 28;    // Añadir nueva clave
$edades["Juan"] = 26;     // Modificar existente
?>
```
- Eliminación de pares:
```php
<?php
unset($edades["María"]);
?>
```
- Recorrido con `foreach`:
```php
<?php
foreach ($edades as $nombre => $edad) {
    echo "$nombre tiene $edad años\n";
}
?>
```
- Funciones útiles:
	- `array_keys($array)` devuelve un array de todas las claves
	- `array_values($array)` devuelve un array de todos los valores
	- `array_search($valor, $array, true)` busca un valor y retorna su clave
	- `asort($array)` y `arsort($array)` ordenan por valor manteniendo la asociación
### Ejemplos prácticos
##### Creación y acceso
```php
<?php

$telefonos = [
  "Ana" => "555-1234",
  "Luis" => "555-5678"
];
echo "Teléfono de Luis: " . $telefonos["Luis"];
// Salida: Teléfono de Luis: 555-5678

?>
```
##### Recorrido con `foreach`
```php
<?php

foreach ($telefonos as $nombre => $numero) {
    echo "$nombre: $numero\n";
}
// Ana: 555-1234
// Luis: 555-5678

?>
```
##### Obtener solo claves o valores
```php
<?php

$claves = array_keys($telefonos);
$valores = array_values($telefonos);
print_r($claves);   // ["Ana", "Luis"]
print_r($valores);  // ["555-1234", "555-5678"]

?>
```
##### Buscar clave por valor
```php
<?php

$clave = array_search("555-1234", $telefonos, true);
echo $clave; // "Ana"

?>
```
##### Ordenar por valor
```php
<?php

$precios = ["manzana"=>1.2, "pera"=>0.8, "uva"=>2.5];
asort($precios);
print_r($precios);
// ["pera"=>0.8, "manzana"=>1.2, "uva"=>2.5]

?>
```
## Notas
---
- Los `arrays` asociativos son ideales para datos con etiquetas semánticas (p. ej. configuración, perfiles)
- Mantener los `arrays` ordenados según su lógica de negocio mejora la legibilidad y evita confusiones.
- Para grandes colecciones, considerar estructuras especializadas (p. ej. `SplObjectStorage`) si las claves deben ser objetos.
