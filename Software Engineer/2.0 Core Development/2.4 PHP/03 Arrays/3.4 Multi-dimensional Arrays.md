Este tema aborda cómo en PHP un array puede contener otros arrays ([[3.1 Arrays]]) como valores, permitiendo estructuras de múltiples niveles. Cada nivel añade un índice más para acceder a sus elementos, y aunque PHP soporta profundidades arbitrarias, en la práctica rara vez se va más allá de 3 niveles por complejidad. Internamente, todos los arrays de PHP son `ordered maps`, por lo que no hay distinción de motor entre arrays indexados, asociativos o multidimensionales. El recorrido de estos arrays se realiza con bucles anidados (generalmente `foreach` ([[5.0 Loops]])), y PHP ofrece la sintaxis literal (`[]`o `array(...)`) para inicializarlos de forma anidada.
### Definición
Un array multidimensional es simplemente un array cuyos valores son a su vez otros arrays. PHP soporta arrays con dos, tres, cuatro o más niveles de profundidad, aunque más de tres suele resultar difícil de gestionar para la mayoría. El número de dimensiones de un array indica cuántos índices necesitas para seleccionar un elemento, p. ej. `$matriz[0][1]` en un array bidimensional.
### Conceptos clave
- `Ordered map` interno: PHP implementa todos los arrays como mapas ordenados, reteniendo el orden de inserción, sea cual sea tu tipo.
- Array bidimensional: Un array de arrays; cada elemento del primer nivel es un array que contiene valores escalares o incluso otros arrays.
- Array tridimensional: Un array cuyos valores son arrays bidimensionales, requiriendo tres índices para acceder a un valor:
- Creación anidada: Usa literales anidados
```php
<?php

$arr2D = [
	[1,2,3],
	[4,5,6]
];
```
- Acceso  mediante múltiples índices:
```php
<?php

echo $arr2D[1][2]; // 6
```
- Recorrido con bucles anidados:
```php
<?php

foreach ($arr2D as $fila) {
  foreach ($fila as $valor) {
    echo $valor;
  }
}
```
- Complejidad práctica: Aunque PHP admite más de 3 dimensiones, conviene limitarse a 2 o 3 para mantener la legibilidad.
### Ejemplos prácticos
##### Array bidimensional (indexado)
```php
<?php

$cars = array(
  array("Volvo",22,18),
  array("BMW",15,13),
  array("Saab",5,2),
  array("Land Rover",17,15)
);
echo $cars[1][0]; // BMW

```
##### Array bidimensional asociativo
```php
<?php

$tbl = [
  "row1" => ["a"=>1, "b"=>2],
  "row2" => ["a"=>3, "b"=>4]
];
echo $tbl["row2"]["b"]; // 4
```
##### Recorrido tabular con `foreach`
```php
<?php

$tbl = [
  [1,2,3],
  [4,5,6],
  [7,8,9]
];

foreach ($tbl as $fila) {
  foreach ($fila as $num) {
    printf("%2d ", $num);
  }
  echo "\n";
}
```
##### Array tridimensional
```php
<?php

$arr3D = [
  [
    [1,0,9],
    [0,5,6]
  ],
  [
    [0,4,6],
    [0,0,1]
  ]
];

echo $arr3D[1][0][2]; // 6
```
##### Iteración de array asociativo multidimensional
```php
<?php

$students = [
  ["name"=>"Ana","age"=>25],
  ["name"=>"Luis","age"=>30]
];

foreach ($students as $st) {
  foreach ($st as $k=>$v) {
    echo "$k => $v; ";
  }
  echo "\n";
}
```
## Notas
---
- Usar arras multidimensionales para representar tablas o matrices de datos en scripts sencillos.
- Para datos muy jerárquicos, considerar usar objetos o `SplObjectStorage` para mayor claridad.
