En PHP las estructuras condicionales (`if`, `elseif` y `else`) ([[4.1 Control Structures]]) permiten ejecutar diferentes bloques de código según la evaluación booleana de expresiones, facilitando el control de flujo en la aplicación. A diferencia de `else`, que se ejecuta siempre que la condición `if` y todos los `elseif` previos sean falsos, `elseif` añade una nueva comprobación que sólo corre si su propia condición es verdadera. PHP acepta tanto la forma combinada `elseif` como `else if` (ambas con comportamiento equivalente al usar llaves), aunque en la sintaxis alternativa basada en dos puntos sólo se admite `elseif` de una sola palabra. Para decisiones sencillas, el operador ternario (`condición ? valor_si_true : valor_si_false`) ofrece una notación abreviada con el mismo fin que un `if…else`. Finalmente, en escenarios de múltiples coincidencias sobre un mismo valor, `switch` ([[4.3 switch]]) suele ser más legible y, en algunos bucles intensivos, algo más eficiente al evaluar la expresión una sola vez.
### Definición
- `if (condicion) {...}`: evalúa la condición; si es `true`, ejecuta el bloque interior.
- `elseif (condicion) {...}`: tras un `if` o `elseif` previo falso, comprueba su propia condición y, si es `true`, ejecuta su bloque.
- `else {...}`: se ejecuta siempre que `if` y todos los `elseif` anteriores hayan sido falsos.
- Equivalencia `elseif` vs `else if`: ambas formas funcionan igual con llaves `{}`, pero sólo la forma de una palabra `elseif` es válida en la sintaxis alternativa con dos puntos (`if...: elseif...: ...endif;`).
### Conceptos clave
- **Orden de evaluación secuencial**: se procesan `if` -> primer `elseif` verdadero -> resto se ignora; si ninguno, `else`.
- **Bloques mutuamente excluyentes**: al cumplirse una condición, los bloques siguientes no se evalúan.
- **Sintaxis alternativa (en plantillas HTML/PHP)**
```php
if (condición): 
    // …
elseif (otra): 
    // …
else: 
    // …
endif;
```
- **Operador ternario** ([[4.4 Null Coalescing Operator]]): `condicion ? expr_true : expr_false` como atajo para `if...else` simple.
- **Rendimiento**: usar `elseif` suele ser más eficiente que múltiples `if` independientes, porque evita reevaluar condiciones previas cuando una se cumple.
### Ejemplos prácticos
### `if...elseif...else` básico
```php
<?php

$a = 5; $b = 3;
if ($a > $b) {
    echo "a es mayor que b";
} elseif ($a == $b) {
    echo "a es igual a b";
} else {
    echo "a es menor que b";
}
```
##### Sintaxis alternativa (plantillas)
```php
<?php if ($usuario == "admin"): ?>
    <p>Acceso total</p>
<?php elseif ($usuario == "editor"): ?>
    <p>Acceso limitado</p>
<?php else: ?>
    <p>Acceso denegado</p>
<?php endif; ?>
```
##### Operador ternario
```php
<?php

$estado = ($edad >= 18) ? "Adulto" : "Menor";
echo $estado;
```
##### Equivalencia `elseif` / `else if`
```php
<?php

if ($x > 0) { echo "Positivo"; }
else if ($x < 0) { echo "Negativo"; } // idéntico a elseif
else { echo "Cero"; }
```
