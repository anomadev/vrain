Los **Named Arguments** fueron introducidos en PHP 8.0 ([[1.1 What is PHP]] para permitir pasar valores a una función o método **basándose en el nombre del parámetro** en lugar de su posición, mejorando la legibilidad y haciendo los argumentos independientes del orden [php.net](https://www.php.net/manual/en/functions.arguments.php?utm_source=chatgpt.com). Esto facilita **omitir** parámetros con valores por defecto sin necesidad de pasar `null` o repetir posiciones intermedias [Tutorials Point](https://www.tutorialspoint.com/php/php_named_arguments.htm?utm_source=chatgpt.com). Se pueden **combinar** con argumentos posicionales, siempre que los posicionales aparezcan primero y los nombrados después . Además, PHP 8.1 amplió su flexibilidad al permitir **trailing commas** en llamadas, lo que hace más limpio el formateo en listados verticales [upsun.com](https://upsun.com/blog/php-8-0-named-arguments/?utm_source=chatgpt.com). Sin embargo, los nombres de parámetro pasan a ser parte de la **API pública**, por lo que renombrarlos puede romper compatibilidad en llamadas existentes [Medium](https://darkghosthunter.medium.com/php-named-arguments-are-your-friends-8076959b2f11?utm_source=chatgpt.com).
### Definición
Los `named arguments` permiten invocar una función o método indicando el parámetro al que corresponde cada valor, con la sintaxis `parameterName: value`. Bajo el capó, PHP asigna el valor al parámetro correspondiente según su nombre, ignorando la posición relativa en la lista de parámetros.
### Conceptos clave
- **Orden independiente:** puedes pasar argumentos en cualquier orden, siempre que el nombre sea correcto [Tutorials Point](https://www.tutorialspoint.com/php/php_named_arguments.htm?utm_source=chatgpt.com).
- **Omisión de valores predeterminados:** permite saltar parámetros con valores por defecto sin pasar `null` ni escribir comas vacías [Tutorials Point](https://www.tutorialspoint.com/php/php_named_arguments.htm?utm_source=chatgpt.com).
- **Mezcla con posicionales:** cualquier argumento posicional **debe** ir antes que uno nombrado en la llamada [Tutorials Point](https://www.tutorialspoint.com/php/php_named_arguments.htm?utm_source=chatgpt.com).
- **`Trailing comma`:** desde PHP 7.3 en llamadas y PHP 8.0 en declaraciones, puedes incluir una coma final para facilitar el formateo [upsun.com](https://upsun.com/blog/php-8-0-named-arguments/?utm_source=chatgpt.com).
- **Errores por nombre incorrecto:** pasar un nombre de parámetro inexistente lanza un `TypeError` en tiempo de ejecución [Blog - stitcher.io](https://stitcher.io/blog/php-8-named-arguments?utm_source=chatgpt.com).
- **Impacto en API pública:** renombrar parámetros rompe las llamadas nombradas existentes, por lo que se considera parte de la firma pública [Medium](https://darkghosthunter.medium.com/php-named-arguments-are-your-friends-8076959b2f11?utm_source=chatgpt.com).
- **Compatibilidad:** no compatible con versiones anteriores a PHP 8.0, evitando su uso si requieres retrocompatibilidad sin capas adicionales [Atatus](https://www.atatus.com/blog/named-arguments-in-php/?utm_source=chatgpt.com).
### Ejemplos prácticos
##### Llamada simple con `Named Arguments`
```php
<?php

function saludar(string nombre, string saludo = "Hola!") {
	echo "$saludo, $nombre";
}

saludar(nombre: "Ana", saludo: "Buenos díás!"); // Buenos días!, Ana

```
##### Omitiendo parámetros con valor por defecto
```php
<?php

saludar(nombre: "Luis"); // Hola!, Luis (saludo se toma por defecto)

```
##### Mezclado posicional y nombrado
```php
<?php
// El argumento posiciones 'Maria' debe ir antes que el nombrado
saludar("Maria", saludo: "Que tal!"); // Que tal!, Maria

```
##### `trailing comma` en la llamada
```php
<?php

saludar (
	nombre: "Pedro",
	saludo: "Buenas!",
);

```
##### Error por nombre inválido
```php
<?php

saludar(usuario: "Carlos"); 
// TypeError: Named parameter $usuario does not exist

```
## Notas
--- 
- Mantener la firma pública estable: evita renombrar parámetros tras su lanzamiento.
- Combinar con `trailing comma` para un formateo vertical limpio en llamadas complejas.
