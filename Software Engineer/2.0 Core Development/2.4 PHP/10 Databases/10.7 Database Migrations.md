Las migraciones de bases de datos ayudan a realizar un seguimiento de los cambios en el esquema de su base de datos, lo que facilita el traslado de una versión de una base de datos a otra.
Las migraciones nos permiten evolucionar el diseño de nuestra base de datos de forma iterativa y aplicar estas actualizaciones en nuestros servidores de desarrollo, ensayo y producción.
Esto puede ahorrar mucho trabajo manual. Pero además, las migraciones mantienen la consistencia en todos los entornos, lo que reduce la probabilidad de comportamientos inesperados.
No existe un mecanismo de migraciones integrado estándar en PHP, pero los frameworks PHP potentes y populares como Laravel tienen soluciones sólidas para las migraciones.
