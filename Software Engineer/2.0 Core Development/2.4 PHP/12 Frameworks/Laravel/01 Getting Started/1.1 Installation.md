Laravel el un framework web de PHP con sintaxis expresiva y enfoque "`developer-first`" que facilita desde proyectos pequeños hasta aplicaciones a escala empresarial. Para instalarlo necesitas tener `PHP 8.4+`, `Composer` y el instalador de Laravel, además de Node.js y `npm` (o `Bun`) para compilar activos frontend. Puedes instalar todo de forma rápida en macOS, Windows o Linux usando los scripts de `php.new` para PHP, el instalador oficial de Composer y luego `composer global require laravel/installer`. Tras ello, creas tu proyecto con `laravel new app-name`, y lo ejecutas con `composer run dev`, lo que levanta el servidor de desarrollo y `Vite` para `assets`. Opcionalmente, Laravel Herd ofrece un entorno gráfico todo en uno para desarrollar en macOS y Windows.
### Definición
La instalación de Laravel consiste en:
- **Preparar el entorno**: Asegurar PHP 8.4+, `Composer` y Laravel Installer instalados globalmente.
- **Instalar dependencias**: Usar `Composer` para descargar el instalador de Laravel y, dentro del proyecto, las librerías del framework.
- **Crear la aplicación**: Ejecutar `laravel new app-name` para scaffolding inicial con tests, bases de datos y frontend incluidos.
- **Levantar el servidor local**: Con `composer run dev` para iniciar PHP y Vite, accediendo después a `http://localhost:8000`.
### Conceptos clave
- **Requisitos previos**
	- **PHP 8.4+**: recomendado instalarlo con `curl -fsSL https://php.new/install/.../8.4`
	- **Composer**: gestor de dependencias de PHP (ver instalación en getcomposer.org)
	- **Laravel Installer**: paquete global de `Composer` (`composer global require laravel/installer`)
	- **Node.js y npm**: para compilar CSS/JS con Vite (o alternativamente Bun)
- **Creación de la aplicación**
	- **Comando**: `laravel new name-app`
	- **Opciones interactivas**: elegir framework de tests, base de datos, starter kit.
- **Servidor de desarrollo**
	- **Script de Composer**: `composer run dev` (inicia servidor en `localhost:8000` y Vite)
	- **Alternativamente "`dump and die`"**: `php artisan serve` para servidor PHP standalone
- **Entornos integrados**
	- **Laravel Herd (macOS/Windows)**: instala PHP, Nginx y herramientas gráficas.
	- **Valet, Homestead y Sail**: otras opciones de desarrollo local.
### Ejemplos prácticos
##### Instalar Laravel Installer vía Composer
```bash
composer global require laravel/installer
```
##### Crear nueva app
```bash
laravel new myApp
```
##### Iniciar entorno de desarrollo
```bash
cd myApp
npm install && npm run build
composer run dev
```
*Accede luego a http://localhost:8000*
##### (Opcional) Instalar y usar Herd en macOS
```bash
curl -fsSL https://herd.laravel.com/install.sh | bash
herd open
```
*Entorno todo en uno con `PHP`, `Nginx` y `Node`*
