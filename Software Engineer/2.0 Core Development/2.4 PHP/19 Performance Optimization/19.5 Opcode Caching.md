> [!info] Opcode Caching (Roadmap SH)
> El almacenamiento en caché de códigos de operación es una técnica que puede mejorar significativamente el rendimiento de PHP. Funciona almacenando el código de bytes de los scripts precompilados en memoria, eliminando así la necesidad de que PHP cargue y analice los scripts en cada solicitud. Para el almacenamiento en caché de códigos de operación, se suele usar la extensión `OPCache` ([[19.2 Caching Strategies]]) en PHP. Con ella, la versión compilada del script PHP se almacena para solicitudes posteriores, lo que reduce la sobrecarga del análisis y la compilación del código. Como resultado, sus aplicaciones experimentan una ejecución más rápida y un menor uso de CPU.
