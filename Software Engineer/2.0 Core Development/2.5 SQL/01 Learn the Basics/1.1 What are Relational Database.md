Una base de datos relacional (RDB) es un sistema para almacenar datos organizados en tablas, que están estructuradas por filas (registros) y columnas (campos). Las relaciones entre estas tablas se defines a través de claves primarias y foráneas, permitiendo mantener la integridad de los datos.

> En otras palabras: Imagina una hoja de cálculo de Excel, donde cada hoja representa una tabla y filas son datos concretos (clientes, productos, etc.). La relación entre hojas se da porque comparte información clave (como un ID de cliente).

### Conceptos Clave
- **Tabla**: conjunto de datos organizados por filas y columnas.
- **Fila (Registro)**: Una entrada individual en la tabla.
- **Columna (Campo)**: Una categoría de datos (ej. nombre, correo, etc.)
- **Clave primaria (`Primary Key`)**: Identificador único de cada fila.
- **Clave foránea (`Foreign Key`)**: Campo que referencia la clave primaria de otra tabla.
- **Relación**: Asociación lógica entre dos tables mediante claves.
- **Integridad referencial**: Garantiza que los datos relacionados entre tablas sean válidos.

Las bases de datos relaciones son un tipo de sistema de gestión de bases de datos (DBMS) que almacena y proporciona acceso a punto de datos relacionados. Basados en el modelo relacional introducido por E.F. Codd en 1970, utilizan una estructura que permite organizar los datos en tables con filas y columnas. Sus características principales incluyen:
- Uso de SQL (leguaje de consulta estructurado) para consultar y gestionar datos.
- Compatibilidad con transacciones ACID (atomicidad, consistencia, aislamiento, durabilidad).
- Aplicación de la integridad de los datos mediante restricciones (por ejemplo, claves primarias, claves externas).
- Capacidad de establecer relaciones entre tablas, lo que permite realizar consultas complejas y recuperar datos.
- Escalabilidad y soporte para entornos multiusuario.
Algunos ejemplos de sistemas de bases de datos relacionales populares incluyen MySQL, PostgreSQL, Oracle y Microsoft SQL Server. Se utilizan ampliamente en diversas aplicaciones, desde proyectos de pequeña escala hasta grandes sistemas empresariales, debido a su confiabilidad, consistencia y potentes capacidades de consulta.
