Laravel Cloud es una plataforma de infraestructura totalmente gestionada para desplegar y escalar aplicaciones Laravel.
Laravel Cloud es la única plataforma como servicio (PaaS) totalmente gestionada y optimizada al máximo para el despliegue y la escalabilidad de aplicaciones Laravel.

- **No se requiere configuración de la aplicación ni del servidor**.
- **Bases de datos MySQL/Postgres totalmente gestionadas y almacenamiento clave-valor compatible con Redis**.
- **Colas administradas**.
- **Despliegues y escalabilidad sin interrupciones**.
- **Escalabilidad a Cero**. Las tareas programadas y los trabajos en cola continúan ejecutándose mientras el entorno está en modo de suspensión.
- **Vista previa de entornos**. Aprovisiona automáticamente un entorno totalmente aislado para cada `pull request`.
- **TLS automático y balanceo de carga**. Todas las aplicaciones reciben certificados TLS, que se actualizan automáticamente. El trafico entrante también se distribuye entre réplicas de aplicaciones disponibles.
- **Cloud Edge Network**. Se asocia con `Cloudflare` para ofrecer una red de seguridad y almacenamiento en caché distribuida globalmente.
- **Monitorización y registros**.

> [!info] PHP 8.2 → PHP 8.5 and Laravel 9.x or greater
