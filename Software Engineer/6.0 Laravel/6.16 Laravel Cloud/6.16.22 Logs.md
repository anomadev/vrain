### Introduction
Laravel Cloud permite ver y buscar los registros de la aplicación Laravel para un entorno determinado a través de la pestana `"Logs"`. Al ver tus registros (`logs`), puedes filtrarlos según el tipo:
- **`Application logs`**: Entradas de registro escritos por la aplicación Laravel, ya sea manualmente o cuando la aplicación encuentra una excepción no controlada.
- **`Access logs`**: Laravel Cloud genera automáticamente entradas de registro cuando la aplicación Laravel recibe una solicitud (`request`).

### Log retention
Laravel Cloud prioriza los registros más recientes y, con el tiempo, eliminará los registros más antiguos cuando superen el período de retención o la cuota de almacenamiento permitida por tu plan (lo que ocurra primero).
