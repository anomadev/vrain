## Definición
Los bucles en `Blade` son directivas que envuelven las estructuras de control iterativas de PHP (`for`, `foreach`, `while`). Su ventaja principal radica en la legibilidad del código y en la inyección automática de una variable de utilidad llamada `$loop`, que proporciona información contextual sobre la iteración actual.

### El estándar de oro: `@forelse`
En el desarrollo profesional, rara vez se usa un `@foreach` simple si los datos provienen de una base de datos. El patrón `@forelse` es la forma más limpia de manejar colecciones que podrían estar vacías sin necesidad de envolver todo en un bloque `@if (count($items) > 0)`.
```php
<ul>
    @forelse ($users as $user)
        <li>{{ $user->name }}</li>
    @empty
        <p>No hay usuarios registrados.</p>
    @endforelse
</ul>
```
### La variable mágica `$loop`
Dentro de cualquier bucle `@foreach` o `@forelse`, Laravel inyecta automáticamente una variable `$loop`. Este objeto permite acceder a metadatos útiles sin tener que gestionar contadores manuales.
**Propiedades más utilizadas**
- `$loop->first`: Devuelve `true` si es la primera iteración. Útil para clases `CSS "active"`.
- `$loop->last`: Devuelve `true` si es la última iteración.
- `$loop->iteration`: El número de la iteración actual (empezando en 1).
- `$loop->index`: El índice de la iteración actual (empezando en 0).
- `$loop->remaining`: Cuántas iteraciones quedan.
- `$loop->count`: El total de elementos en la colección.
- `$loop->parent`: Cuando hay bucles anidados, permite acceder al objeto `$loop` del nivel superior.
```php
@foreach ($users as $user)
    <div @class(['first-item' => $loop->first, 'last-item' => $loop->last])>
        {{ $user->name }} ({{ $loop->iteration }} de {{ $loop->count }})
    </div>
@endforeach
```
### Directivas de control `@break` y `@continue`
`Blade` permite interrumpir o saltar iteraciones basándose en condiciones, manteniendo la lógica dentro de la directiva para evitar bloque de código PHP pesados.
```php
@foreach ($users as $user)
    {{-- Salta los usuarios inactivos --}}
    @continue($user->is_inactive)

    <li>{{ $user->name }}</li>

    {{-- Detiene el bucle si llegamos a 10 elementos --}}
    @break($loop->iteration == 10)
@endforeach
```
### Bucles tradicionales
Aunque menos comunes en las vistas (que suelen iterar sobre colecciones), `Blade` soporta los bucles clásicos:
- `@for`: Para iteraciones basadas en un contador numérico.
- `@while`: Itera mientras se cumpla una condición booleana.
```php
@for ($i = 0; $i < 10; $i++)
    El valor es {{ $i }}
@endfor
```

## Analogía
Imagina que un bucle es una Línea de Ensamblaje.
- `@foreach` es la cinta transportadora que pasa cada pieza una por una.
- `@forelse` es la cinta con un sensor inteligente: si no detecta piezas, activa automáticamente un aviso de "fuera de stock" (`@empty`).
- `$loop` es el Panel de Control del operario: le dice exactamente qué número de pieza tiene en las manos, si es la primera del lote o cuántas faltan para terminar la caja.

## Buenas Prácticas
1. **Preferir `@forelse` sobre `@foreach`**: Siempre que los datos provengan de una fuente externa, asume que la colección podría estar vacía.
2. **Evitar lógica pesada**: Si necesitas filtrar datos dentro del bucle (ej. `@continue($user->age < 18)`), considera si ese filtrado debería haberse hecho en el `Query Builder` del controlador para ahorrar memoria.
3. **Uso de `$loop->parent`**: En bucles anidados muy profundos, el uso de `$loop->parent` puede ser confuso. Si la lógica es muy compleja, considera extraer el bucle interno a un `Blade Component`.
